## 监控网页视觉变化并通过邮件和 Telegram 发送通知

这是一个完整的指南，旨在帮助您部署一个 Python 脚本。该脚本可以自动化监控指定网页的视觉变化（例如商品补货），并在检测到变化时，通过电子邮件和 Telegram 即时发送通知。

### 核心功能

*   **自动化监控**：使用 Selenium 驱动无头 Chrome 浏览器，定期访问并截取网页。
*   **视觉对比**：通过 Pillow (PIL) 库对比新旧截图的像素差异，判断页面是否发生实质性变化。
*   **双重通知**：变化一旦确认，会同时通过 `msmtp` 发送邮件和 Telegram Bot API 发送即时消息。
*   **服务器适用**：脚本和环境配置都针对在无图形界面的服务器（如 Docker 容器、VPS）上运行进行了优化。
